# 06 · 變更紀錄

← 回 [手冊目錄](./README.md)

變更紀錄頁（`/history`）保存資料的異動歷史，分兩種模式，用頁首切換。

> 目前此頁**所有登入者都可查看**（不限管理者）。

---

## 一、匯入紀錄

查每一次 xlsx 匯入改動了什麼。

```mermaid
flowchart LR
    A[左側：匯入批次清單] --> B[點選某一批]
    B --> C[右側：該批的變更明細<br/>依學員分組]
    C --> D[看每位學員<br/>逐欄「舊值→新值」]
```

| 區塊 | 說明 |
|---|---|
| 左側清單 | 每次匯入一列：檔名、時間，以及 **+新增 / ↻更新 / 未變** 的數量標籤 |
| 右側明細 | 選一批後，依學員分組顯示，每位標「新增/更新」，並列出逐欄的舊值→新值 |
| 名稱搜尋 | 在明細中依姓名過濾 |

---

## 二、手動編輯

查每一筆在表格上「直接編輯儲存格」的紀錄（見 [01 學員管理](./01-學員管理.md)）。

| 欄位 | 意義 |
|---|---|
| 時間 | 何時改的 |
| 學員 | 改了哪位學員 |
| 欄位 | 改了哪個欄位 |
| 舊值 → 新值 | 從什麼改成什麼 |
| **操作者** | 是誰改的（登入者帳號/姓名） |

- 顯示最近 500 筆，可依學員姓名搜尋。
- 這是所有就地編輯的稽核軌跡，方便追查「這個資料是誰、什麼時候改的」。

---

## 三、兩種紀錄的差別

| | 匯入紀錄 | 手動編輯 |
|---|---|---|
| 來源 | xlsx 批次匯入 | 表格上逐格編輯 |
| 粒度 | 一次匯入（多筆） | 單一儲存格 |
| 有記操作者? | 記匯入這個動作 | ✅ 記到「誰改的」 |
| 對應章節 | [02 匯入匯出](./02-匯入匯出.md) | [01 學員管理](./01-學員管理.md) |

> 想查「帳號管理類」的操作（開帳號、停用、重設密碼、資料匯出…），那屬於**操作稽核**，在另一個頁面，見 [07 帳號管理與稽核](./07-帳號管理與稽核（管理者）.md)。
