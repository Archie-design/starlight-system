# 05 · 儀表板與心之使者

← 回 [手冊目錄](./README.md)

這兩個頁面把資料彙整成統計圖表，方便一眼掌握整體狀況。資料範圍都限你所屬體系（系統管理者可切換星光/太陽）。

---

## 一、儀表板 `/dashboard`

### KPI 卡片（四個）

| 卡片 | 意義 |
|---|---|
| 系統總學員數 | 目前體系全部學員數 |
| 已過期會籍 | 會籍已過期的人數 |
| 30 天內到期會籍 | 即將到期的人數 |
| 有組別學員數 | 已歸入關懷長分組的人數 |

### 圖表

| 圖表 | 內容 | 可互動? |
|---|---|---|
| **各階上課人數** | 各課程階段（一階～五階、五運）的上課人數長條圖 | — |
| **各組人數統計** | 每個關懷長分組的人數（橫向長條） | — |
| **課程付款狀態分布** | 各階段的付款狀態堆疊圖（已完款 / 部分付款 / 退款完成） | ✅ 點某段 → 跳出該階段/狀態的學員清單 |

> **各階上課人數是「獨立計數」**，不是漏斗（階段不必然照順序完成），所以各階人數不代表逐階遞減。

### 清單與匯出

| 區塊 | 說明 |
|---|---|
| **會籍到期預警清單** | 已過期 + 30 天內到期的學員表，每位可點進學員；右上有 **匯出 Excel** 鈕（此為此頁專屬的小型匯出） |
| **未完款學員** | 有報名課程但付款狀態非「已完款」的學員（含未完款的階段/備註，最多 100 筆） |

> 圖表或清單中的學員姓名多可點擊，會帶你回學員管理頁並自動搜尋該人（`/students?search=…`）。

## 二、心之使者專區 `/spirit`

聚焦「心之使者」——有「心之使者加入日」的學員，依「心之使者組別」分組，年資來自「累積年資」。

### KPI 卡片（四個）

| 卡片 | 意義 |
|---|---|
| 心之使者總數 | 目前體系的心之使者人數 |
| 組別數 | 有幾個心之使者小組 |
| 平均年資 | 平均累積年資（以月呈現） |
| 無組別人數 | 有加入但沒分到組的人數（大於 0 時以琥珀色提醒） |

### 圖表

| 圖表 | 內容 | 可互動? |
|---|---|---|
| **年資分佈** | 依累積年資分桶的長條圖 | — |
| **各組人數** | 每組人數（橫向長條） | ✅ 點某條 → 跳出該組成員清單 |
| **各組平均年資** | 每組平均年資（月，橫向長條） | 滑鼠移上顯示該組人數 |

### 資料品質提醒

三份清單，幫你找出需要修正的資料（姓名可點回學員頁修正）：

| 清單 | 意義 |
|---|---|
| 有加入日但無組別 | 已是心之使者卻沒分到組 |
| 無累積年資 | 缺年資資料 |
| 單人小組 | 只有一人的組（可能需併組或補人） |

> 系統管理者的頁首有星光/太陽切換，切換後會重新統計。
