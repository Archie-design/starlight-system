## 名詞速查

| 名詞 | 說明 |
|---|---|
| **體系** | 系統把學員分成兩個體系：**星光** 與 **太陽**。多數功能只在你所屬的體系內作用。 |
| **角色** | 你的權限層級：系統管理者 / 體系長 / 體系管理者（詳見下表）。 |
| **會籍狀態** | 已過期 / 30 天內到期 / 90 天內到期 / 有效 / 無資料。 |
| **課程階段** | 一階～五階，另有五運與特殊課程。 |
| **區域** | 北區 / 中區 / 南區。 |

## 角色權限總表

系統共有三種角色（括號內為畫面上顯示的名稱）：

| 功能 / 範圍 | 系統管理者<br>(superadmin) | 體系長<br>(system_admin) | 體系管理者<br>(admin) |
|---|:---:|:---:|:---:|
| 綁定體系 | 無（跨體系） | 單一體系 | 單一體系 |
| 可見資料範圍 | 星光 + 太陽 | 只有自己體系 | 只有自己體系 |
| 切換體系（星光/太陽） | ✅ | ❌ | ❌ |
| 學員管理 / 篩選 / 編輯 | ✅ | ✅ | ✅ |
| 匯入 / 匯出 xlsx | ✅ | ✅ | ✅ |
| 關懷長專區 / 管理分組 | ✅ | ✅ | ✅ |
| 資料維護專區 | ✅ | ✅ | ✅ |
| 儀表板 / 心之使者 | ✅ | ✅ | ✅ |
| 變更紀錄 | ✅ | ✅ | ✅ |
| **帳號管理**（/admin/users） | ✅ 全部帳號 | ✅ 只限同體系、不可建系統管理者 | ❌ 會被導回學員頁 |
| **登入紀錄 / 操作稽核** | ✅ | ✅ | ❌ 會被導回學員頁 |

> **怎麼變成體系長 / 體系管理者？**
> 關懷長以上的學員，第一次用「學員 ID + 手機末四碼」自助登入時，系統會自動建立帳號：
> **體系長 / 體系長共同經營 → 體系長（system_admin）**；**關懷長 / 關懷長共同經營 → 體系管理者（admin）**。
> 詳見 [00 快速上手](./00-快速上手.md)。
